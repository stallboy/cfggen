# 项目概述

## 项目背景

cfggen 是一个基于 Java 的配置生成工具，专门用于从多种数据源（CSV、Excel、JSON）生成多语言代码。项目采用 Gradle 构建，支持 Java 21，旨在提供高效、灵活的配置代码生成解决方案。

## 核心功能

### 1. 多数据源支持
- **CSV文件**：支持GBK编码，自动检测BOM头
- **Excel文件**：支持.xlsx格式，使用FastExcel库高效读取
- **JSON文件**：支持结构化配置数据

### 2. 多语言代码生成
- **Java**：生成Java数据类和代码，支持sealed接口
- **C#**：生成C#代码和字节码
- **TypeScript**：生成TypeScript代码和Schema
- **Go**：生成Go语言代码
- **Lua**：生成Lua脚本，支持内存优化

### 3. 复杂结构支持
- **嵌套结构**：支持任意深度的结构嵌套
- **多态接口**：支持接口和实现类定义
- **递归结构**：支持递归数据结构的定义
- **表格映射**：支持auto、pack、sep、fix、block等多种映射方式

### 4. 数据关系管理
- **主键系统**：支持简单主键、复合主键、多字段主键
- **唯一键**：支持多种访问方式的唯一键定义
- **外键引用**：支持主键引用、唯一键引用、列表外键
- **枚举入口**：支持枚举表和入口表的定义

### 5. 国际化支持
- 多语言术语检查和验证
- 语言切换支持
- 术语兼容性检查

### 6. 工具功能
- 数据验证和搜索
- 二进制到文本转换
- Schema格式转换（XML到CFG）
- 编辑器服务器支持
- AI集成支持

## 项目目标

1. **高效性**：快速处理大量配置数据
2. **灵活性**：支持多种输入格式和输出语言
3. **可扩展性**：易于添加新的生成器和数据源
4. **易用性**：提供丰富的工具和验证功能

## 技术特点

- **模块化设计**：清晰的模块划分，便于维护和扩展
- **模板引擎**：使用JTE模板引擎生成代码
- **性能优化**：支持缓存和性能分析
- **错误处理**：完善的错误报告和堆栈跟踪

## 使用场景

- 游戏配置数据管理
- 多平台应用配置生成
- 国际化项目配置处理
- 自动化代码生成工作流

## 项目状态

当前项目处于活跃开发状态，支持多种主流编程语言的代码生成，并持续优化性能和功能。
