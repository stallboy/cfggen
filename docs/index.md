# 📊 策划配置解决方案

![intro](assets/intro.png)

包含三个核心系统

1. 🗂️ 配表系统
   - 定义结构，生成代码

2. 🎨 编辑器
   - 浏览和编辑复杂数据

3. 🤖 AI 集成
   - MCP 服务，AI 生成配置

## 💡 建议

1. 📈 表格型的数据就用excel来做

    - 表格型数据里可能包含子结构的列表，一般的关系数据库需要引入另外的表来做‘规范化’，
      这里不推荐这样做，而推荐使用[映射到表格]一节介绍的机制来在同一个表里直接配置。
    - 可能包含一些简单的多态结构，如果不复杂，推荐直接嵌入的方式。

2. 🏗️ 对于复杂的多态数据用json来存，用cfgeditor.exe来编辑

    - 用一个json表示单个复杂数据，比如技能effect，技能buff
    - 多个json通过相互引用表示图结构，比如对话，任务流程

## 🚀 扩展功能

- **🤖 AI 集成** - MCP 服务支持 AI 助手交互，AI 生成配置
- **🔌 VSCode 插件** - 语法高亮、智能提示、快速导航
- **💻 多语言支持** - Java、C#、TypeScript、Go、Lua、JSON
