# 🖼️ 预览资源

预览资源功能允许在编辑器中查看和管理与数据记录关联的多媒体资源。

## 🔗 关联资源

每个数据记录（record）都可以关联一个或多个资源文件。这些资源会在编辑器中以预览形式展示，方便用户快速查看和管理。

### 资源查找模式

系统支持两种资源查找模式：

#### 1. 显式配置模式
- **工作原理**：通过配置明确指定资源表（如 `asset.assets`）
- **适用场景**：资源信息存储在数据库表中，有明确的结构化数据
- **示例**：配置指向 `asset.assets` 表，系统会从该表中查找关联的资源信息

#### 2. 隐式约定模式
- **工作原理**：基于文件命名约定自动发现资源
- **文件命名格式**：`[tableName]_[recordId].[扩展名]`
  - `tableName`：数据表名
  - `recordId`：记录ID
  - `扩展名`：文件类型（如 jpg、mp4、png 等）
- **适用场景**：资源文件存储在特定目录中，通过命名约定与数据记录关联
- **示例**：`users_123.jpg` 表示 users 表中 ID 为 123 的记录关联的图片

## 👁️ 资源展示

### 支持的文件类型
系统支持预览以下类型的资源文件：

| 类型 | 图标 | 说明 |
|------|------|------|
| **视频** | 🎬 | MP4、AVI、MOV 等视频格式 |
| **音频** | 🔊 | MP3、WAV、AAC 等音频格式 |
| **图片** | 🖼️ | JPG、PNG、GIF 等图片格式 |
| **字幕** | 📝 | SRT、ASS 等字幕文件 |
| **其他** | 📄 | 其他类型的文件 |

### 资源统计信息

在编辑器节点上会显示资源统计信息，格式为：`1v2a2o`

**统计标识说明**：
- `v`：视频文件数量（video）
- `a`：音频文件数量（audio）
- `i`：图片文件数量（image）
- `o`：其他文件数量（other）

**示例**：
- `1v2a2o`：表示有 1 个视频、2 个音频、2 个其他文件
- `3i1v`：表示有 3 个图片、1 个视频

### 操作指南

#### 查看详细资源列表
1. 点击节点上的统计信息按钮（如 `1v2a2o`）
2. 系统会弹窗显示所有关联资源的详细信息
3. 列表中包含每个资源的：
   - 文件类型图标
   - 文件名
   - 完整路径

#### 快速访问资源文件
- **点击资源路径**：在资源列表中点击任意资源的路径
- **系统操作**：会自动打开文件资源管理器（Explorer）并选中对应的文件
- **使用场景**：快速定位和打开资源文件进行编辑或查看
